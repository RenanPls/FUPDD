## Ação

Dado um número inteiro, faça uma função que verifique se ele é primo.  

## Entrada

* Um numero inteiro N.

### Saida

* 1 caso N for primo e 0 caso contrário.

## Exemplos

``` txt
>>>>>>>>
5
========
1
<<<<<<<<

>>>>>>>>
21
========
0
<<<<<<<<

>>>>>>>>
2
========
1
<<<<<<<<
```
